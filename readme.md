# TucilStima 1 - Brute Force Word Search Puzzle Solver


## General Information
A word search puzzle solver made in Java, implemented with brute force algorithm.


## Technologies Used
- Java language

## Dependencies
- Java 17


## Features
- Solving word search puzzles

## How to run?
### For windows users:
- `cd Tucil1_13520016`
- `./build.bat`
- `./run.bat`

### Manual Build and Run
- `javac -d bin/ src/*.java`
- `cd bin && java Main`
